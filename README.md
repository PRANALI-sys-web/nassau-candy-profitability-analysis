# Nassau Candy Product Line Profitability Analysis

## Project Overview
This project analyzes product line profitability and margin performance for Nassau Candy Distributor using Data Science and Financial Analytics.

## Objectives
- Identify top profitable products
- Analyze profit margin by region
- Study relationship between units sold and profit

## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit
- Jupyter Notebook

## Files in this Repository
- clean_nassau_candy_data.csv – Cleaned dataset used for analysis
- Nassau_Candy_Profitability_Analysis.ipynb – Data analysis and visualization notebook
- app.py – Streamlit dashboard application
