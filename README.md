# Nassau Candy Product Line Profitability Analysis

## Project Overview
This project focuses on analyzing product line profitability and margin performance for Nassau Candy Distributor using data science and visualization techniques. It helps in identifying high-performing and underperforming products based on key financial metrics like sales, cost, gross profit, and margin percentage.

## Objectives
Identify top profitable products
Analyze profit distribution across regions
Study relationship between units sold and profit
Perform Pareto analysis (80/20 rule)
Build an interactive dashboard for insights

## 📊Dashboard Features
✔ Key Performance Indicators (Sales, Profit, Margin %)
✔ Top 10 Profitable Products
✔ Profit by Region
✔ Units vs Profit Analysis
✔ Division-wise Profit & Margin
✔ Pareto Analysis (80% Contribution)
✔ Cost vs Sales Analysis
